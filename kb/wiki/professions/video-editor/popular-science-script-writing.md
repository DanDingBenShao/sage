# 科普视频文案脚本技巧

## 两种核心叙事结构

### 结构 A：悬疑叙事（适合长视频/纪录片风）

```
开篇悬念（前 15s）
  └── 用具体年份+事件开头：1953年，一个科学家做了个疯狂的决定...

历史背景
  └── 交代时代、地点、学术背景

人物描写
  └── 突出科学家特质：坚持/天才/偏执

实验细节
  └── 用数字描写方法、数量、时间

发现与突破
  └── 明确科学结论，强调重要性

意义与影响
  └── 解释如何改变学界/社会

结尾反思/悬念
  └── 提出深思问题或引出下集
```

### 结构 B：反直觉文案（适合短视频/抖音）

```
对比戳人（前3s）
  └── 为什么别人躺赚你却越忙越穷？

抛反常识观点
  └── 赛道错了，努力全白费

3-4 条底层逻辑
  └── 每条都配金句，第二人称"你"

真实案例 x1-3
  └── 反差要大的案例

升维对比
  └── 拉高格局，对比不同思维方式

具体行动建议 x3
  └── 改变认知的方法
```

## 语言技巧

| 原则 | 正确 | 错误 |
|------|------|------|
| 类比降维 | "房间不收拾会变乱"讲熵增 | "熵是系统混乱度的度量" |
| 口语化 | "两个粒子隔千里也能心灵感应" | "量子纠缠是量子力学中的非定域性现象" |
| 短句 | 拆长句为短句，15-25字一句 | 40+字学术复合句 |
| 第二人称 | "你知道吗，你的身体里..." | "人体内存在..." |

## 字数控制

| 视频时长 | 文案字数上限 | 要点数量 |
|---------|-----------|---------|
| 1-3 分钟（抖音） | 600-1000 字 | 1-2 个 |
| 3-6 分钟（B站短科普） | 不超过 1800 字 | 2-3 个 |
| 7-10 分钟（B站长科普） | 2000-3500 字 | 3-5 个 |
| 15-20 分钟（深度） | 4000-6000 字 | 5-8 个 |

## DeepSeek 文案提示词模板

```
我是一名短视频科普博主，请你帮我写一篇约[时长]分钟的科普短视频文案，要求:

1. 风格：[历史+科学+故事化]，口播感强、语言生动、引人入胜
2. 结构：
   - 开篇悬念：用[年份]+[科学事件]开头，引发好奇
   - 历史背景：交代时代、地点、学术背景
   - 人物描写：突出科学家的特质
   - 实验细节：用数字描写方法、数量、时间
   - 发现突破：明确科学结论
   - 意义影响：解释如何改变学界/社会
   - 结尾反思：提出深思问题
3. 语言：长短句结合、口语化、多用比喻、避免术语堆砌
4. 多使用数字、年份、数量增强真实感
```

## 脚本标注规范

在脚本中标注以下要素，方便后期：
- `[B-roll]` — 需要插入素材的位置
- `[动画]` — 需要做动画的概念
- `[大字]` — 关键结论叠加大字
- `[音效]` — 需要音效强调的位置
- `[转场]` — 段落切换点
