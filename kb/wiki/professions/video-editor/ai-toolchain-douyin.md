# 抖音科普 5步AI工具链

## 工具链总览

```
[Step 1] DeepSeek → 文案
[Step 2] DeepSeek → 分镜表格
[Step 3] 豆包 AI → 生图（9:16）
[Step 4] 即梦 Seedance 2.0 → 图生视频
[Step 5] 剪映 → 剪辑合成+配音+字幕+BGM
```

## 详细步骤

### Step 1: DeepSeek 生成文案

```
提示词模板：
我是一名短视频科普博主，请你帮我写一篇约8分钟的科普短视频文案，要求:
1. 风格：历史+科学+故事化，口播感强、语言生动
2. 结构：开篇悬念→历史背景→人物描写→实验细节→发现突破→意义影响→结尾反思
3. 语言：长短句结合、口语化、多用比喻、避免术语堆砌
```

### Step 2: DeepSeek 拆分镜

```
追问提示词：
请根据这个脚本，帮我生成详细的分镜头描述，
每个镜头要包含画面内容、时长、镜头运动方向。
要求：3D动画风格，加入主人公形象，画面与文案对应。
```

### Step 3: 豆包 AI 生图

- 打开豆包 → 图像生成功能
- 逐条粘贴分镜描述
- **关键：选择 9:16 竖屏比例**
- 不满意重新生成（免费）

### Step 4: 即梦 Seedance 2.0 图生视频

- 上传图片到即梦
- 输入动态指令示例：
  - "镜头缓慢推进，聚焦到显微镜"
  - "画面从左向右平移"
  - "烛光闪烁，背景微微虚化"
- 提示词越简单越好

### Step 5: 剪映合成

1. 导入所有视频片段，按分镜排列
2. 加转场（淡入淡出/闪白）
3. 配音：点击【文本】→【添加口播稿】→ 粘贴文案 → 选音色 → 【智能分割】→【添加到轨道】
4. 加字幕：选择模板后【全局应用】
5. BGM：舒缓纯音乐，音量调低不压人声
6. 导出：1080P 或 4K

## 工具对比

| 环节 | 抖音轻量方案 | 大圆镜专业方案 |
|------|------------|-------------|
| 文案 | DeepSeek | DeepSeek/Gemini 多模型交叉 |
| 生图 | 豆包 AI（免费） | Midjourney（付费） |
| 图生视频 | 即梦 Seedance 2.0 | 即梦 + 可灵 |
| 配音 | 剪映内置 | ElevenLabs |
| 剪辑 | 剪映 | 剪映 |
| 审核 | 无 | 专家审核 |

## 适用场景

- 抖音科普短视频（1-3 分钟）
- 零基础小白快速入门
- 低成本批量生产
- 个人创作者单兵作战
