# 科普视频选题策略

## 核心原则

**七分选题，三分制作** — 选题决定了视频流量的天花板。

## 五大选题来源

### 1. 猎奇+硬核
- 一听就想点进来的主题
- 例：电击复活实验、记忆移植、基因编辑婴儿、深海生物
- 关键词：实验、复活、移植、改造、禁忌

### 2. 生活现象切入
- 从日常"为什么"出发
- 例：泡面桶为什么不能进微波炉、熬夜真的会变傻吗
- 降低心理门槛，人人都有共鸣

### 3. 网络热梗/热点
- 借势当前热门话题
- 例：结合热门电影讲对应科学原理
- 天然带搜索流量

### 4. 反直觉认知
- 打破常识的冲击感
- 例："你以为的常识，其实全错了"
- 模式：对比戳人 → 抛反常识观点 → 逻辑拆解 → 案例佐证

### 5. 争议话题
- 有讨论度的话题自带传播
- 注意把握好度，客观呈现多个角度

## 系列化策划

- 按学科脉络规划系列（如《生命科学简史》《大脑简史》）
- 从热门话题入手 → 逐步深入冷门深度内容
- 满足观众"追番"心理，提高回访率

## 粉丝共创

- 评论区收集观众"点菜"
- 投票决定下期选题
- 开发 App 让粉丝投票（大圆镜模式）
- 好处：选题即验证需求，粉丝参与感=传播力

## 选题验证清单

- [ ] 一句话能说清主题吗？
- [ ] 目标观众看了标题会想点进来吗？
- [ ] 现有知识能支撑深度讲解吗？
- [ ] 有 2-3 个可展开的知识点吗？
- [ ] 是否适合做成系列？
