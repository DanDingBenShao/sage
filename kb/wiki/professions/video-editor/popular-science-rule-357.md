# 科普视频 3-5-7 法则

## 核心结构

```
[0-3s]  钩子（Hook）
   ├── 悬念式："你每天刷牙的方式，可能都是错的"
   ├── 反常识式："其实，太阳并不是黄色的"
   ├── 承诺式："给我一分钟，让你成为朋友圈里的天文大神"
   └── 视觉冲击式：开场就是酷炫实验画面

[3s-5min]  主体展开
   ├── 避免知识堆砌，每次只讲 2-3 个知识点
   ├── 每 60 秒必须切换画面/视觉元素（"视觉锚点"）
   └── 用类比降低门槛，拒绝纯口播

[最后7s]  记忆点
   ├── 一句话总结核心观点
   ├── 制造"哇"时刻
   └── 引导互动（弹幕/评论/关注）
```

## 视觉锚点规则

- 每 60 秒至少切换一次画面
- 切换方式：B-roll / 动画 / 图表 / 文字卡片 / 场景变换
- 目的：防止观众注意力流失，避免"听播客"式体验

## 适用平台

| 平台 | 调整 |
|------|------|
| B站 知识科普 | 5-8min，允许留白，深度展开 |
| 抖音 科普 | 1-3min，每 2-4s 切换镜头 |
| YouTube 教程 | 10-20min，章节标记辅助跳转 |

## 常见错误

- ❌ 开头 30 秒还在铺垫背景 → 观众已划走
- ❌ 中间 3 分钟画面不变 → 纯口播完播率极低
- ❌ 结尾突然结束 → 没有给观众"转发/评论"的理由
