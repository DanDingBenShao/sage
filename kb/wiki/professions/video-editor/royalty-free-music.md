## 免费方案推荐

```
Uppbeat (uppbeat.io):
  免费计划: 无限下载，需在视频描述中注明 (attribution)
  付费 ($7/月): 免 attribution
  推荐: 独立创作者首选

Pixabay Music (pixabay.com/music):
  完全免费，无需 attribution
  质量参差不齐，需要耐心挑
  推荐: 快速找到"够用"的 BGM

YouTube Audio Library:
  在 YouTube Studio 中直接访问
  完全免费，Safe for YouTube
  推荐: 只发 YouTube 的创作者
```

## BGM 选择清单

```
情绪 → 视频类型映射:
  激昂/史诗 → 产品发布/品牌片
  轻快/Lo-fi → Vlog/日常/教程
  温暖/钢琴 → 情感叙事/回忆杀
  电子/科技感 → 软件教程/科技评测
  紧张/悬疑 → 问题引入/转折前
  舒缓/氛围 → 冥想/自然/留白段落
  
技术检查:
  [ ] BPM 与视频剪辑节奏匹配 (120-140 BPM = 快节奏)
  [ ] 有明确的起承转合结构 (方便分段切换)
  [ ] 下载 WAV/FLAC 格式 (不用低码率 MP3 再编辑)
  [ ] 确认版权许可 (attribution 还是免标注)
```

## 版权避坑

1. **网易云/QQ 音乐下载的不行** — 那些是个人收听许可，不覆盖视频商用
2. **"无版权音乐" ≠ 真的无版权** — 确认来源平台明确标注 Royalty-Free
3. **AI 生成音乐的版权灰色地带** — Suno/Udio 生成的音乐版权归属仍在争议中，商用谨慎
4. **CC 协议要看清** — CC BY (需署名) vs CC BY-NC (禁止商用) vs CC0 (完全自由)
