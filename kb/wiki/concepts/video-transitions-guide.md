# 视频转场完全指南

## 核心原则

> **转场是讲故事的工具，不是装饰品。** 如果你解释不了"这里为什么用这个转场"，那就去掉它。

## 转场类型速查

| 转场 | 视觉 | 含义 | 适用 | 时长 |
|------|------|------|------|------|
| **硬切 (Cut)** | 瞬间切换 | 连续性、同一时空 | 对话/教程/90%的常规编辑 | 0s |
| **淡入淡出 (Fade)** | 渐黑/渐亮 | 开始/结束/章节 | 视频开头结尾/段落分割 | 0.5-2s |
| **交叉溶解 (Dissolve)** | 渐变叠加 | 时间流逝/回忆/情绪 | 蒙太奇/情绪转折/时间跳转 | 0.5-1.5s |
| **滑动 (Slide/Push)** | 推入推出 | 方向性/递进 | 演示/列表/对比展示 | 0.3-0.8s |
| **缩放 (Zoom)** | 放大/缩小 | 聚焦/冲击力 | 强调关键信息/快节奏切换 | 0.2-0.5s |
| **擦除 (Wipe)** | 边缘扫过 | 场景切换/风格化 | 科幻/品牌/复古风格 | 0.5-1s |
| **J-cut** | 声音先入 | 听觉引导视觉 | 场景自然过渡/制造期待 | 0.5-2s |
| **L-cut** | 声音延续 | 画变声不变 | 反应镜头/场景余韵 | 0.5-2s |

## 转场决策矩阵

```
你需要:
  正常切镜头? → 硬切 (永远的第一选择)
  表示时间过去了? → 交叉溶解
  章节结束/开始? → 淡入淡出
  从 A 话题转到 B 话题? → 滑动 + 标题卡
  强调一个关键信息? → 缩放 + 音效
  制造"下一秒会发生什么"的期待? → J-cut
  让观众看到角色的反应? → L-cut
```

## J-cut / L-cut (最重要但最被忽视的转场)

```
J-cut (音频先行):
  [画面: A 在讲话] ────────┤                    ├── [画面: 下一个场景]
  [音频: A 在讲话] ────────┼── [音频: B 开始说话] ──┼── [音频: B]
                              ↑
                         声音先切入, 画面还在 A

L-cut (音频延续):
  [画面: 展示结果] ─────┤                          ├── [画面: 下一个场景]
  [音频: 讲解结果] ────┼──────────────────────┼── [音频: 讲解]
                       ↑
                  画面已切走, 声音还在继续
```

J-cut 制造期待感，L-cut 制造沉浸感。两者让转场"感觉不到"。

## 各平台转场风格指南

| 平台 | 推荐转场 | 避免 |
|------|---------|------|
| YouTube 教程 | 硬切 90% + 偶尔溶解 | 花哨转场 |
| B站 知识科普 | 硬切 + 滑动卡片 + 淡入淡出 | 过度缩放 |
| 抖音/短视频 | 快切 + 缩放冲击 + 闪光 | 慢溶解 |
| Vlog | 硬切 + J-cut/L-cut | 炫技转场 |
| 品牌/商业片 | 硬切 + 滑动 + 擦除 | 随意乱用 |

## 常见误区

| 误区 | 正确做法 |
|------|---------|
| 每个切点都换一种转场 | 全片用 2-3 种转场, 保持一致性 |
| 转场时长 > 2s | 最多 1.5s, 长溶解会让观众觉得"卡了" |
| 用了转场但没留 handle | 剪辑时在切点前后多留 0.5-1s 素材用于转场叠加 |
| 画面转了但声音硬切 | 至少加 2-3 帧的音频交叉淡化 |
| 转场比内容更抢眼 | 如果观众记住你的转场而不是你的内容, 你输了 |

## ffmpeg 转场命令

```bash
# 交叉溶解 0.5s
ffmpeg -i clip1.mp4 -i clip2.mp4 -filter_complex \
  "[0:v]settb=AVTB,fps=30[v0]; \
   [1:v]settb=AVTB,fps=30[v1]; \
   [v0][v1]xfade=transition=fade:duration=0.5:offset=D1[outv]; \
   [0:a][1:a]acrossfade=d=0.5[outa]" \
  -map "[outv]" -map "[outa]" output.mp4
# offset = clip1 的时长 - 0.5 (转场开始的时间点)

# 滑动 (从右推入)
xfade=transition=wiperight:duration=0.5:offset=D1

# 缩放
xfade=transition=zoomin:duration=0.3:offset=D1
```
