# 截图与标注工具

## 工具对比

| 工具 | 平台 | 价格 | 杀手功能 |
|------|------|------|---------|
| **Snipaste** | Win/Mac | 免费 | F1 截图 → F3 贴到屏幕上，截图标注后可直接对比 |
| **CleanShot X** | macOS | $29 一次性 | 滚动截图、录屏转 GIF、隐藏桌面图标后截图 |
| **ShareX** | Windows | 免费开源 | 截图+录制+标注+上传一体化 |
| **Greenshot** | Windows | 免费 | 简单轻量，适合只截图+标注的场景 |

## Snipaste 使用技巧

```
F1: 截图 (自动检测窗口/区域边界)
F3: 将截图"贴"在屏幕上 (浮窗，无视窗口层级)
  → 非常适合"这里和那里对比"的教程场景
Ctrl+滚轮: 调整贴图透明度
双击贴图: 关闭
Shift+F1: 截图时自动贴到屏幕
```

## 标注视觉规范

**一致性比花哨重要。** 选定一套标注风格后全视频统一：

```
颜色:
  强调色: #FF0000 (红) — 用于箭头、框
  辅助色: #FFD700 (黄) — 用于文字高亮背景
  模糊: 高斯模糊 8-12px — 用于遮挡敏感信息

线条:
  箭头: 实线 3px，箭头末端三角形
  框: 圆角矩形，线宽 3px
  不要: 虚线、锯齿线、发光边框

文字:
  标注文字: 无衬线加粗，字号 ≥ 16px，白字黑底框
  位置: 被标注元素旁边，不要遮挡关键内容
```

## 截图增强流程

```
原始截图 → [1] 裁剪到关键区域 → [2] 添加箭头/框标注
        → [3] 模糊敏感信息 → [4] 添加编号/文字说明
        → [5] 导出 PNG (不要 JPG — 文字和线条会糊)
```

## 在视频中使用截图

- 截图分辨率 ≥ 视频分辨率（1080p 视频用 ≥ 1920x1080 截图）
- 截图放大展示局部时用"放大镜"效果（圆形遮罩放大 2x）
- 截图停留时间 ≥ 3 秒（观众需要时间读）
- 多张图对比时并排显示，加分隔线或编号
